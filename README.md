### Init Project

**1-** Clone repository.

**2-** Install dependecies.
```js
yarn install
```

**3-** Run Development environment.
```js
yarn run dev
```
**3-** Run unit test (for **backend** only).
```js
yarn run test
```

**5-** Build (for **frontend** only)
```js
yarn run build
```

**6-** Run built application (for **frontend** only)
```js
yarn run start
```
